#Question :- Write a Python program to square the elements of a list using map() function.
             Sample List:    [4, 5, 2, 9]
             Square the elements of the list:     [16, 25, 4, 81]

#Solution :- 

func =[4, 5, 2, 9]
result = map(lambda  x:x**2,func)
print(list(result))


#Solution :- [16, 25, 4, 81]
